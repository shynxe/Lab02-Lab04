# Lab2-4
 
## 6. Cofetarie

Creati o aplicatie care permite gestiunea stocului de materii prime folosite intr-o cofetarie.

Fiecare materie prima din stoc are un nume unic, un producator si cantitate.

Aplicatia permite:

 * adaugare de materie prima (daca exista deza in stoc se actualizeaza cantitatea)

 * modificare materie prima 

 * stergere  materie prima

 * Vizualizare materii prime care satisfac un criteriu (nume incepe cu o litera data, cantitatea disponibila e mai mica decat un numar dat)

 * Vizualizare materii prime din stoc ordonat dupa nume sau cantitate disponibila (crescator/descrescator)
